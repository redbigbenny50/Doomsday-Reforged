Reforged

Doomsday

A NeoForge-based rebuild of the Doomsday Decoration mod, targeting modern Minecraft (1.21.1) with updated tooling and fixes.

MC 1.21.1 NeoForge 21.1.x Java 21

Overview

Reforged Doomsday is a modernization pass on Doomsday Decoration. The goal is to keep the original content while making it buildable and stable on current NeoForge versions.

Requirements

  • Java 21 (Eclipse Adoptium recommended)
  • Minecraft 1.21.1
  • NeoForge 21.1.x

Setup

  1. Clone the repo.
  2. Set Java 21 for Gradle (see below).
  3. Build or run the dev client.
$env:JAVA_HOME = "C:\Path\To\jdk-21"
$env:Path = "$env:JAVA_HOME\bin;$env:Path"

Build / Run

Compile

.\gradlew compileJava

Dev Client

.\gradlew runClient

Build Jar

.\gradlew build

Troubleshooting

  • If Gradle complains about Java 8, your shell is not using Java 21.
  • If blocks are non-solid, check collision/shape settings in the code.
  • Use a clean test world when validating fixes.
